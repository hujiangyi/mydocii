
    dB 是一个纯计数单位
    
    
    功率与电压、电流的关系是P=V2/R=I2R

增益是一个输出和输入的功率比率

    
    
    对于功率，dB = 10*log()。对于电压或电流，dB = 20*log()
    
    
    dB的意义其实再简单不过了，就是把一个很大（后面跟一长串0的）或者很小（前面有一长串0的）的数比较简短地表示出来
    
    
    X = 100000 = 10*log(10^5) = 50 dB
    
    
    X = 0.000000000000001 = 10*log(10^-15) = -150 dB
